# Hao Wu

> **Backend engineer who builds reliable data systems. Reducing software entropy.**

### About

Backend software engineer with 15 years of experience, working remotely with US teams. I started in **backend architecture and legacy modernization**, and over time moved toward the data side of the stack — building **ELT pipelines, dimensional models, and reproducible analytics** on top of the systems I help design.

I like work where correctness is verifiable: typed code, tested transformations, and pipelines anyone can clone and run.

### What I do

* **Data engineering** — ELT pipelines, dimensional modeling, and orchestration; turning raw sources into clean, tested, queryable warehouses.
* **Backend architecture** — decoupling monoliths into services and modernizing legacy stacks (Python 2 → 3, JS → TS) with zero downtime.
* **Engineering standards** — type safety, executable data-quality tests, and documentation that lets distributed teams move fast without breaking things.

### Selected project

* **[qdev](https://github.com/haowu0802/qdev)** — a reproducible market-forecast evaluation pipeline: Python ingestion with visible data-quality checks, DuckDB as a local warehouse, and dbt for staging plus tests on grain and nullability. Forecasts are scored with proper scoring rules (Brier, calibration) rather than treated as trading signals.
